# Notebooks de Análisis

Esta carpeta contiene Jupyter notebooks para análisis exploratorio y documentación del proceso.

## Notebooks

### 001_exploracion_inicial.ipynb
Exploración inicial de los datos y estadísticas descriptivas.

### 002_limpieza_datos.ipynb
Proceso de limpieza y transformación de datos.

### 003_analisis_ods4.ipynb
Análisis específico para ODS 4 (Educación de calidad).

### 004_analisis_genero.ipynb
Análisis de brechas de género (ODS 5).

### 005_desigualdades.ipynb
Análisis de desigualdades regionales (ODS 10).

## Uso

```bash
# Activar entorno
source .venv/bin/activate

# Iniciar Jupyter
jupyter lab
```
